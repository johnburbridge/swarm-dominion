# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Swarm Dominion, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email security concerns to [INSERT EMAIL] with:

- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Suggested fix (if any)

We will acknowledge your report within 48 hours and provide updates as we work on a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

Security updates will be provided for the current release.

## Scope

This security policy applies to:

- The game client
- The dedicated server software
- Official multiplayer services (when available)

Third-party forks and modifications are not covered by this policy.

## Responsible Disclosure

We appreciate security researchers who follow responsible disclosure practices. We will:

- Acknowledge your contribution (unless you prefer to remain anonymous)
- Work with you to understand and resolve the issue
- Not pursue legal action for good-faith security research
